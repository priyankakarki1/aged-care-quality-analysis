# Australian Aged Care Quality Analysis

## Overview
As someone who has worked in the aged care industry for over 2 years, I wanted to combine my real-world experience with my data analytics skills to analyse the quality of aged care services across Australia. This project uses the Australian Government's star ratings data to uncover insights about what makes aged care facilities perform better or worse.

Data source: Australian Government GEN Aged Care Data (February 2026)
2,315 providers analysed across all Australian states and territories.

## Key Findings

1. Tasmania leads all states with the highest average star rating
2. Government-operated facilities outperform private providers with an average rating of 4.25 stars
3. Small facilities deliver better quality care than larger ones, averaging 4.05 stars
4. NSW has the most 5-star providers with 119 top-rated facilities
5. Rural and remote facilities surprisingly outperform major cities, challenging the common assumption that city care is superior

## Tools Used

- Python (Pandas, Matplotlib, Seaborn) - data cleaning and visualisation
- SQL (MySQL) - data analysis and querying
- Power BI - interactive dashboard

## Project Structure

- data/ - cleaned dataset
- notebooks/ - Python scripts for cleaning and visualisation
- sql/ - SQL queries used for analysis
- charts/ - chart images generated from Python
- dashboard/ - Power BI dashboard file

Having worked in aged care myself, I could relate to the data on a personal level and understand what the numbers actually mean on the ground.
